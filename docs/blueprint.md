# **App Name**: CIS Sovereign

## Core Features:

- Live Election Dashboard: An administrative tool to monitor total votes cast and real-time class turnout using Supabase subscriptions.
- Intelligent Candidate Management: Admin panel to manage reorderable positions and candidates with photo uploads stored in cloud storage.
- Population-Enforced Voting: Prevents over-voting by checking the submitted vote against a fixed class population cap in the database.
- One-Shot Voting Session: Secures the electoral process using session tokens and browser flagging to ensure each student only votes once.
- Real-time Leaderboards: A public results page that updates live as votes are recorded, highlighting current front-runners for every position.
- Electoral Insights Tool: Generative AI powered tool that analyzes turnout trends to identify engagement patterns or suggest scheduling for peak voting periods.
- Responsive Voting Portal: Mobile-optimized card-based interface for candidate selection with a final summary verification step.

## Style Guidelines:

- Primary color is Teal (#1A7A6E) to signify professionalism and institutional trust, paired with Navy Blue (#1A3A5C) for high-contrast navigation components.
- Accent color: Amber (#F5A623) used strategically for call-to-action buttons, 'Leading' badges, and critical turnout highlights.
- Background: Clean light grey (#F5F5F5) for page containers with a pure white surface for content cards.
- Body and Heading font: 'Inter'. Bolder weights are utilized for headings to establish a clear hierarchy. Note: currently only Google Fonts are supported.
- Functional icons using thin line weights to maintain a minimalist, sophisticated aesthetic.
- A sidebar-based layout for Admin tools and a focused, center-aligned single-column layout for the mobile voting experience.
- Subtle slide-and-fade transitions between the candidate selection and the final submission screen to provide a fluid user experience.
